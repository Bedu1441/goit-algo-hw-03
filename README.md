# goit-algo-hw-03

Домашнє завдання до теми «Рекурсивні функції, алгоритми та приклади їх застосування»*

## Структура репозиторію

- `task1_recursive_sort/main.py` — рекурсивне копіювання файлів з вихідної директорії, сортування за розширеннями.
- `task2_koch_snowflake/main.py` — візуалізація фракталу «сніжинка Коха» з вибором рівня рекурсії.
- `task3_hanoi_tower/main.py` — (додатково) розв'язання задачі «Ханойські башти» з логуванням кроків.

## Як запускати

### Завдання 1

```bash
cd task1_recursive_sort
python main.py path/to/source path/to/dest
# або
python main.py path/to/source
# у цьому випадку буде створено директорію "dist" поруч із скриптом
